Полное руководство по ChetBot

Добро пожаловать в официальную документацию ChetBot! Эта документация создана для того, чтобы помочь вам настроить бота на вашем сервере, использовать его веб-дашборд и максимально раскрыть его потенциал.

1. Введение

Что такое ChetBot?

ChetBot — это многоцелевой Discord-бот премиум-класса, разработанный для того, чтобы полностью автоматизировать рутину на вашем сервере. Подобно передовым решениям (таким как ProBot, Dyno или Carl-bot), мы сделали упор на Web Dashboard — мощную панель управления в браузере, благодаря которой вам больше не нужно запоминать сотни команд.

Главные преимущества:

  • 🌐 Универсальный Дашборд: Настраивайте всё через красивый и понятный интерфейс (React + Vite).
  • ⚡ Высокая производительность: Бэкенд на aiohttp гарантирует, что изменения на сайте мгновенно применяются на вашем сервере без перезагрузок.
  • 🛡️ Продвинутая защита: Инструменты модерации, включая режим "Lockdown" (Изоляция) и анти-спам, защитят ваш сервер от рейдов.
  • 🎫 Встроенные Тикеты (CTD): Поддержка пользователей через приватные ветки (Threads) с авто-закрытием неактивных обращений.
  • 🏆 Турнирные сетки: Уникальный модуль для генерации сеток "Single Elimination".
  • 🎨 Embed Builder: Создание шикарных Embed-сообщений без знания JSON.

2. Начало работы с Дашбордом

Взаимодействие с ChetBot на 90% происходит через веб-дашборд.

Как войти в панель?

  1. Перейдите по ссылке вашего дашборда (например, cheterin.online).
  2. Нажмите кнопку "Login with Discord".
  3. Вы будете перенаправлены на официальную страницу авторизации Discord. Разрешите приложению доступ к вашему профилю.
  4. После успешного входа вы попадете на главную страницу управления сервером.
ℹ️ Кто имеет доступ?
Бот автоматически проверяет ваши права на сервере. Только пользователи с правами администратора (или специально указанными ролями) могут изменять настройки.

Навигация по панели

В левом меню вы найдете все необходимые модули:

  • Overview / Config: Базовые настройки, выбор каналов для логов.
  • Members: Список участников и массовая выдача ролей.
  • Embed Builder: Создание сообщений.
  • Modules: Включение и настройка отдельных функций (Анти-спам, Приветствия и т.д.).
  • Brackets: Управление турнирами.

3. Модули: Модерация и Логи

ChetBot предлагает надежную систему модерации для поддержания порядка.

🛡️ Анти-Спам (Spam Protection)

Модуль автоматически анализирует поток сообщений на сервере.

  • Как работает: При обнаружении флуда (слишком много сообщений за короткое время от одного пользователя), бот удаляет эти сообщения.
  • Наказание: Нарушителю автоматически выдается роль, указанная в переменной SPAM_LOG_ROLE_ID (обычно это роль Muted, запрещающая отправку сообщений).
  • Исключения: В дашборде вы можете указать каналы в SPAM_EXCEPTION_CHANNELS. В этих каналах бот не будет наказывать за спам (удобно для каналов с бот-командами).

🛑 Временные баны (Tempban)

Вместо перманентной блокировки, вы можете дать пользователю время на раздумья.

  • Выдайте временный бан через панель или команду.
  • Бот зафиксирует время снятия бана.
  • Как только срок истечет, бот самостоятельно разбанит пользователя и отправит отчет в лог-канал TEMPBAN_CHANNEL_ID.

🔒 Изоляция сервера (Lockdown)

В случае массовой атаки (рейд ботов) используйте этот режим.

  • Введите команду /antispam или включите Lockdown в панели.
  • Бот мгновенно переопределит права для роли @everyone во всех текстовых каналах, запретив отправку сообщений.
  • После окончания рейда, вы отключаете режим, и права возвращаются в исходное состояние.

📝 Журнал аудита (Audit Logging)

ChetBot фиксирует всё! Выберите канал LOG_CHANNEL_ID в панели конфигурации, и бот будет отправлять туда:

  • Удаленные сообщения (с содержимым).
  • Измененные сообщения (До / После).
  • Входы и выходы участников.
  • Действия модераторов (выдача ролей, баны, закрытие тикетов).

4. Модули: Взаимодействие (Roles & Welcome)

👋 Приветствия (Welcome)

Встречают по одежке. Сделайте так, чтобы каждый новый участник чувствовал себя особенным.

  1. Зайдите в дашборд -> Welcome.
  2. Выберите канал для приветствий (WELCOME_CHANNEL_ID).
  3. Настройте текст сообщения (поддерживаются переменные, такие как упоминание пользователя {user} или количество участников {member_count}).
  4. Бот также может отправлять прощальные сообщения при выходе человека с сервера.

🤖 Автороли (AutoRoles)

Избавьтесь от ручной выдачи базовых ролей.

  • Перейдите в раздел AutoRoles.
  • Выберите одну или несколько ролей из списка.
  • Теперь, как только пользователь присоединится к серверу, ChetBot мгновенно выдаст ему эти роли.

🎭 Роли по реакциям (Reaction Roles)

Создайте меню самовыдачи ролей.

  1. В дашборде перейдите в Reaction Roles.
  2. Укажите ID сообщения (вы можете создать его заранее через Embed Builder).
  3. Привяжите эмодзи к конкретной роли.
  4. Готово! Пользователь ставит реакцию — бот выдает роль. Убирает реакцию — бот забирает роль.

5. Модули: Тикеты и Обратная связь

🎫 Система обращений (CTD - Contact The Developer)

Идеально для серверов, где игрокам нужна приватная поддержка администрации.

Установка панели:

  1. Администратор использует слеш-команду /ctd_setup в специальном текстовом канале (например, #поддержка).
  2. Бот отправит красивое Embed-сообщение с кнопкой "Создать Тикет".

Как это работает:

  • Пользователь жмет кнопку.
  • Бот создает приватную ветку (Thread).
  • В эту ветку автоматически приглашается создатель и все пользователи с ролью поддержки (CTD_ROLE_ID).
  • Никто другой не видит переписку!

Авто-управление (Умное закрытие):

  • Если в тикете тишина 48 часов, бот пишет предупреждение: "Тикет неактивен..."
  • Если тишина длится еще 24 часа — бот автоматически закрывает и архивирует ветку.
  • Модератор может закрыть тикет мгновенно нажатием красной кнопки внутри ветки.

6. Уникальные функции (Эксклюзив)

🏆 Турнирные сетки (Brackets)

Если ваш сервер проводит киберспортивные или фановые турниры, ChetBot заменит вам сторонние сайты вроде Challonge.

  1. Создание: В дашборде (раздел Brackets) вы нажимаете "Создать сетку".
  2. Участники: Добавляете список участников (ники игроков или названия команд).
  3. Генерация: Бот автоматически строит сетку Single Elimination с правильным сидированием (алгоритм обхода Byes для нечетного числа игроков).
  4. Управление матчами: Администратор кликает по победителю матча в браузере, и он проходит в следующий раунд.
  5. Публичная ссылка: Бот генерирует Public URL, который вы можете скинуть игрокам. Они будут видеть турнирную таблицу в реальном времени (без прав на ее изменение).

🎨 Embed Builder (Конструктор сообщений)

Создавайте потрясающие анонсы, правила сервера или гайды прямо в браузере.

  • Интерфейс WYSIWYG: Вы видите изменения в реальном времени.
  • Возможности: Настройка цвета полоски (HEX), добавление автора, иконок, заголовка, тела текста, изображений (Thumbnail и Large Image) и футера.
  • Интеграция: Можно добавлять кнопки к сообщению.
  • Отправка: Выберите канал в выпадающем списке и нажмите "Отправить". Сообщение появится в Discord от имени бота мгновенно.

👥 Массовая выдача ролей (Mass Role Assignment)

Нужно наградить 100 участников ивента ролью?
В разделе Members выберите нужных пользователей галочками, нажмите "Выдать роль" и выберите роль. ChetBot запустит фоновую задачу, которая обойдет ограничения Discord API и безопасно выдаст роли всем выбранным людям, сгенерировав отчет по итогу.

7. Список команд бота (Commands)

ChetBot спроектирован так, чтобы минимизировать использование команд в чате. Тем не менее, некоторые функции удобнее вызывать из Discord:

Команда Описание Требуемые права
/ctd_setup Развернуть панель создания тикетов (CTD) в текущем канале. Управление сервером
/antispam Быстрый переключатель режима защиты/изоляции ролей. Управление сервером
/userinfo Выводит подробную сводку о пользователе (даты регистрации, входа, роли). -

8. Для разработчиков (Self-Hosting)

Хотите запустить ChetBot на своем оборудовании? Следуйте этим шагам.

Требования

  • Python 3.10+ (Рекомендуется 3.11)
  • Node.js LTS (Для фронтенда)

Шаг 1. Клонирование и зависимости

git clone https://github.com/Nanda070/Cheterin_Bot_Dashboard.git
cd Cheterin_Bot_Dashboard
python -m venv venv
source venv/bin/activate  # на Windows: venv\Scripts\activate
pip install -r requirements.txt

Шаг 2. Конфигурация (.env)

Создайте файл .env в корне проекта:

BOT_TOKEN=Твой_Токен_Из_Discord_Developer_Portal
GUILD_ID=ID_Сервера
DISCORD_OAUTH_REDIRECT_URI=http://localhost:8080/api/auth/discord/callback
Внимание! Обязательно добавьте http://localhost:8080/api/auth/discord/callback в список Redirect URIs в настройках вашего бота на портале Discord! Без этого вход в дашборд не будет работать.

Шаг 3. Сборка дашборда

cd dashboard/frontend
npm install
npm run build
cd ../..

Шаг 4. Запуск

python main.py

Откройте браузер и перейдите по адресу http://localhost:8080.

(Подробные инструкции по деплою на Linux Ubuntu (VPS) с использованием Nginx и Certbot находятся в файле DEPLOY.md в корне репозитория).
